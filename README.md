# AS3-obfuscator

Это написанный на PHP5 и построенный на регулярных выражениях (функциях preg_*) простой обфускатор для кода классов на ActionScript 3. 

Он заменяет имена классов и методов в этих классах на случайные строки для запутывания чтения логики сценариев.


## Как это работает

- Кладем структуру (все классы, с папками) в proc/source. 

- Запускаем index.php.

- В случае успешного обфусцирования получаем трехмерную структуру пакет/класс/метод на выводе (для дебаггинга, по большей части), плюс, готовые файлы в proc/destination.


### Примечания

- Если имя класса (и его конструктор) не нужно обфусцировать, например, некий стартовый класс (обычно, Main.as) инициализированный из *.fla, то прописываем его в массив в качестве второго аргумента (в index.php):

	$aIgnoreWords = array_merge(AS3Validator::getReservedWords(), array("Main"));

потом это хозяйство вынесу в отдельный файл.

- Пока автоматическая очистка папки proc/destination не производится, поэтому, при часто измененяемых файлах *.as (из proc/source), в таблице замены используется простой хэш md5, вместо случайного значения.
