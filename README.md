# AS3-obfuscator

Это написанный на PHP5 и построенный на регулярных выражениях (функциях preg_*) простой обфускатор для кода классов на ActionScript 3. 

Он заменяет имена классов и методов во всех файлах *.as на случайные строки для запутывания чтения логики сценариев.


## Как это работает

1. Кладем структуру директории классов в proc/source.

2. Запускаем index.php.

3. В случае успешного обфусцирования получаем трехмерную структуру пакет/класс/метод на выводе (для дебаггинга, по большей части), плюс, готовые файлы в proc/destination.


#### Что необходимо учесть 

1. Если имя класса (и его конструктор) не нужно обфусцировать, например, некий стартовый класс (обычно, Main.as) инициализированный из *.fla, его нужно добавить в массив $aIgnoreWords (потом это хозяйство вынесу в отдельный файл):

		$aIgnoreWords = array_merge(AS3Validator::getReservedWords(), array("Main"));
  
2. Пока автоматическая очистка папки proc/destination не производится, поэтому, при часто измененяемых файлах *.as (из proc/source), в таблице замены используется простой хэш md5, вместо случайного значения.
